# C++ ML Notebook — House Price Prediction

A small, dependency-free machine learning pipeline written in plain C++
(no libraries, no frameworks) that trains a linear regression model with
batch gradient descent and produces a Kaggle-style submission file.

## Files

| File                    | Purpose                                                              |
|--------------------------|-----------------------------------------------------------------------|
| `ml_notebook.cpp`        | The whole "notebook": CSV loader, linear regression, train/predict/write |
| `train.csv`              | 60 labeled rows: `id, area_sqft, bedrooms, age_years, price_k`         |
| `test.csv`               | 20 unlabeled rows: `id, area_sqft, bedrooms, age_years` (no price)     |
| `sample_submission.csv`  | Shows the required output format (`id, price_k`) with placeholder 0.0 |
| `submission.csv`         | Generated by the program — the model's real predictions               |

The data is synthetic, generated from a known relationship
(`price ≈ 50 + 0.12·area + 15·bedrooms − 0.8·age + noise`) so you can sanity
check that the model recovers something close to it.

## Build & run

```bash
g++ -O2 -std=c++17 ml_notebook.cpp -o ml_notebook
./ml_notebook
```

This will:
1. Load `train.csv`
2. Standardize features and fit a linear regression by gradient descent
   (prints training RMSE every 1000 epochs)
3. Print the learned equation in original (unstandardized) units
4. Load `test.csv`, predict `price_k` for each row
5. Write `submission.csv` in the same format as `sample_submission.csv`

## Extending it

- Swap in your own `train.csv` / `test.csv` (keep the same column-name
  pattern, or adjust the `col_index(...)` calls in `main()`).
- The `LinearRegression` class is self-contained — you could extend it with
  L2 regularization (ridge), polynomial features, or swap gradient descent
  for the closed-form normal equation.
- To evaluate properly, split `train.csv` into a train/validation subset
  before fitting (currently it trains on all 60 rows).
